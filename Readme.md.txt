SQL Ecommerce Analysis – Brazilian E-Commerce Dataset (Olist)
Project Overview
-----------------------
This project analyzes the Brazilian E-Commerce Public Dataset by Olist using MySQL.

The goal of this project was to perform end-to-end SQL analysis on real-world e-commerce data and generate business insights related to:

1.Customer behavior
2.Revenue trends
3.Delivery performance
4.Product categories
5.Seller performance
6.Payments
7.Order lifecycle

The complete dataset was imported into MySQL and analyzed using SQL queries.

Dataset Information
------------------------
Dataset: Brazilian E-Commerce Public Dataset by Olist

Step 3 – SQL Analysis
